# UniDB Database Project

A university database project with frontend and backend components.

## Project Structure

```
Uni Database Project/
├── backend/          # Node.js/Express API server
├── frontend/         # React/Vite frontend application
├── Uni_DB.sql        # Database schema file
└── run-app.bat       # Application startup script
```

## Getting Started

### Prerequisites
- Node.js
- SQL Server

### Installation

1. Clone the repository
2. Install backend dependencies:
   ```bash
   cd backend
   npm install
   ```
3. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```

### Running the Application

Use the provided `run-app.bat` script, or:

1. Start the backend server:
   ```bash
   cd backend
   npm start
   ```

2. Start the frontend development server:
   ```bash
   cd frontend
   npm run dev
   ```
